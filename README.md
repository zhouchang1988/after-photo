# 照片整理工具 - After Photo

一个智能的照片整理工具，自动按文件类型分类、检测重复照片并选择最佳质量版本。

**当前版本：v1.0**

## 功能特性

- **按文件类型拆分目录**：自动将 JPG、RAW、视频文件分别整理到对应目录
- **重复照片检测**：使用 pHash 算法检测并归类相似的照片
- **最佳照片选择**：基于清晰度和亮度分布自动选择每组重复照片中的最佳质量照片
- **RAW 文件同步**：自动同步 RAW 文件的标记状态
- **删除多余RAW文件**：删除没有对应JPG文件的RAW文件（需手动确认）
- **自动清理空目录**：步骤3完成后自动删除空的jpg/raw目录
- **日志记录**：自动在工作目录创建带时间戳的日志文件，记录所有操作和输出
- **跨平台支持**：支持 macOS、Windows
- **高效处理**：按时间顺序处理图片，使用30秒时间阈值智能跳过不相关的时间段，大幅提升处理效率
- **时间统计**：显示任务总耗时，方便了解处理进度

## 安装和使用

### 从源码编译

```bash
# 克隆项目
git clone <repository-url>
cd after_photo

# 使用 build.sh 跨平台编译
./build.sh

# 编译并运行测试
./build.sh --test
```

### 运行程序

程序已预编译在 `bin/` 目录下，可根据系统选择对应版本直接使用。

**方式 1：在当前目录运行并输入目标路径**

```bash
# macOS
./bin/after_photo_mac

# Windows
bin\after_photo.exe
```

**方式 2：复制到目标目录运行**

将编译好的程序复制到要整理的照片目录，在该目录下运行。

## 使用步骤

程序提供四个执行步骤：

- `[1]` 按文件类型拆分目录
- `[2]` 检测并归类重复照片
- `[3]` 在重复照片中选择最佳
- `[4]` 删除多余的RAW文件（需确认）

可以组合输入，如 `123` 执行步骤1-3，或 `4` 单独执行步骤4。

**日志记录**：确认工作目录后，程序会自动在该目录创建日志文件 `after_photo_YYYYMMDDHHMMSS.txt`，记录所有操作输出和时间戳，便于后续查阅。

## 工作流程

### 步骤 1：按文件类型拆分目录
- 扫描输入目录中的所有文件
- 创建 `jpg/`、`raw/`、`video/` 目录
- 移动对应扩展名的文件到相应目录

### 步骤 2：检测并归类重复照片
- 按文件创建时间排序扫描 JPG 文件
- 使用 30 秒时间阈值优化处理，大幅减少不必要的哈希计算
- 使用 pHash 算法计算图片相似度
- 相似照片移动到同一子目录
- 同步按照JPG分组结构整理RAW文件

### 步骤 3：在重复照片中选择最佳
- 计算每张照片的质量得分
- 将最佳照片重命名，添加 `-` 后缀
- 自动同步 RAW 文件的标记状态
- 自动删除空的 jpg/ 和 raw/ 目录（无图片文件时）

### 步骤 4：删除多余的RAW文件
- 检查每个 RAW 文件是否有对应的 JPG 文件
- 显示待删除文件列表，需用户确认后执行

## 算法说明

### pHash (Perceptual Hash)
将图片缩小到 32x32 像素，转换为灰度图，执行 2D DCT 变换，提取 8x8 低频系数，生成 64 位哈希，使用汉明距离比较（阈值 ≤ 17）。

### 图片质量评分
- **清晰度**：使用拉普拉斯算子检测边缘信息
- **亮度分布**：分析图片中心和整体亮度比例

## 注意事项

- 最佳照片不会删除其他照片，而是通过添加 `-` 后缀标记
- RAW 文件的标记会与 JPG 文件同步
- 程序默认使用程序所在目录作为输入目录
- 确认工作目录后会自动创建日志文件，记录所有操作
- 执行完成后可选择继续其他步骤、整理其他文件夹或退出程序
- 建议在整理前备份重要照片
