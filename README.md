# After Photo

连拍摄影师的效率工具——自动去重、智能选优，让整理照片不再拖垮创作节奏。

人像、婚礼、野生动物……一次拍摄动辄数百上千张，连拍是必要的，但整理是痛苦的。After Photo 就是帮你从海量重复片中快速筛出最优解，把该留的标出来，把该清的归好类，让你把时间花在修图而不是选图上。

## 核心功能

- **按文件类型拆分**：JPG、RAW、视频自动归入 `jpg/`、`raw/`、`video/` 目录，一步分拣
- **智能去重分组**：pHash 感知哈希 + 30 秒时间阈值，连拍照片自动归组，不用人工一张张对比
- **自动选优**：基于清晰度和亮度分布评分，每组重复片自动标记最佳那张，RAW 同步标记
- **RAW 清理**：一键清理没有对应 JPG 的 RAW 文件，释放存储空间
- **操作留痕**：自动生成带时间戳的日志文件，每一步操作可追溯

## 适合谁

- **人像摄影师**：一次拍摄多角度多表情，连拍几十张只留最自然的一张
- **婚礼摄影师**：全天候跟拍，仪式环节不敢漏按，后期面对上千张重复片
- **野生动物摄影师**：高速连拍捕捉瞬间，回来后要在一堆糊片中找锐的那张
- **运动/赛事摄影师**：关键动作连拍，需要快速选出最清晰的一帧

## 安装和使用

### 从源码编译

```bash
# 克隆项目
git clone <repository-url>
cd after-photo

# 跨平台编译
./build.sh

# 编译并运行测试
./build.sh --test
```

### 运行程序

程序已预编译在 `bin/` 目录下，可根据系统选择对应版本直接使用。

**方式 1：在当前目录运行并输入目标路径**

```bash
# macOS
./bin/after-photo-mac

# Windows
bin\after-photo.exe
```

**方式 2：复制到目标目录运行**

将编译好的程序复制到要整理的照片目录，在该目录下运行。

## 使用步骤

程序提供四个执行步骤：

- `[1]` 按文件类型拆分目录
- `[2]` 检测并归类重复照片
- `[3]` 在重复照片中选择最佳
- `[4]` 删除多余的RAW文件（需确认）

可以组合输入，如 `123` 执行步骤1-3，或 `4` 单独执行步骤4。不输入则默认执行1-3。

## 工作流程

### 步骤 1：按文件类型拆分目录
- 扫描输入目录中的所有文件
- 创建 `jpg/`、`raw/`、`video/` 目录
- 移动对应扩展名的文件到相应目录

### 步骤 2：检测并归类重复照片
- 按文件创建时间排序扫描 JPG 文件
- 使用 30 秒时间阈值优化处理，大幅减少不必要的哈希计算
- 使用 pHash 算法计算图片相似度
- 相似照片移动到同一子目录
- 同步按照 JPG 分组结构整理 RAW 文件

### 步骤 3：在重复照片中选择最佳
- 计算每张照片的质量得分
- 将最佳照片重命名，添加 `-` 后缀标记
- 自动同步 RAW 文件的标记状态
- 自动删除空的 jpg/ 和 raw/ 目录

### 步骤 4：删除多余的RAW文件
- 检查每个 RAW 文件是否有对应的 JPG 文件
- 显示待删除文件列表，需用户确认后执行

## 算法说明

### pHash (Perceptual Hash)
将图片缩小到 32x32 像素，转换为灰度图，执行 2D DCT 变换，提取 8x8 低频系数，生成 64 位哈希，使用汉明距离比较（阈值 ≤ 17）。

### 图片质量评分
- **清晰度**：使用拉普拉斯算子检测边缘信息
- **亮度分布**：分析图片中心和整体亮度比例

## 注意事项

- 最佳照片不会删除其他照片，而是通过添加 `-` 后缀标记
- RAW 文件的标记会与 JPG 文件同步
- 程序默认使用程序所在目录作为输入目录
- 确认工作目录后会自动创建日志文件，记录所有操作
- 执行完成后可选择继续其他步骤、整理其他文件夹或退出程序
- 建议在整理前备份重要照片
